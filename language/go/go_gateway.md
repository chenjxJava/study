快速体验：

```dockerfile
// 开箱即用：
docker pull fagongzi/gateway:v3.0.0-RC

docker run -d --rm -p 80:80 -p 9093:9093 -p 9092:9092  fagongzi/gateway:v3.0.0-RC
```

 浏览器访问 UI：http://127.0.0.1:9093/ui/index.html
